# supreme-waddle-ocr

Simple python script that reads the text from an image

## Running

```shell
make run
```

Desired output will look something like:

```shell
{'text': u'Isporticos Transfer Corp.\n128 Bakeland Ave\nMi...}
```

## Linting

```shell
make lint
```
